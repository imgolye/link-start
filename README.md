#SpringLink 微服务开发平台

#工程结构
``` 
link-tool
├── link-core-boot -- 业务包综合模块
├── link-core-cloud -- cloud封装模块
├── link-core-develop -- 代码生成封装模块
├── link-core-launch -- 基础启动模块
├── link-core-log -- 日志封装模块 
├── link-core-mybatis -- mybatis拓展封装模块 
├── link-core-oss -- 对象存储封装模块 
├── link-core-secure -- 安全模块 
├── link-core-swagger -- swagger拓展封装模块 
├── link-core-test -- 单元测试封装模块 
├── link-core-tool -- 单元测试封装模块 
└── link-core-transaction -- 分布式事物封装模块 
```
